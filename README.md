## About

NotchAI is a macOS AI assistant designed to live in the system notch/menu bar, providing quick access to multiple AI providers such as Claude, ChatGPT, and others through a unified interface.

This project is currently in its early stages and is being developed primarily as a learning project to explore Swift, macOS application development, and integrations with AI platforms and APIs. Features, architecture, and implementation details may change significantly as the project evolves.
